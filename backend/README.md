# 🌿 AgriInsight AI - Backend

## Overview
The backend is developed using Flask and TensorFlow. It loads the trained AI model, processes uploaded leaf images, predicts plant diseases, and returns treatment recommendations through REST APIs.

## Technologies Used
- Python
- Flask
- TensorFlow
- NumPy

## Features
- Image Upload API
- AI Disease Prediction
- Confidence Score
- Treatment Recommendation
- Weather Recommendation
- REST API

## Installation

```bash
cd backend
pip install -r requirements.txt
python app.py
```

The backend runs at:

```
http://localhost:5000
```
